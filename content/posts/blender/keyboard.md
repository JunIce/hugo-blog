# blender 常用快捷键






| 快捷键    | 操作                             |
| --------- | -------------------------------- |
| A         | 全选                             |
| AA        | 取消全选（按两次 A）             |
| Shift +D  | 复制                             |
| G         | 移动                             |
| S         | 缩放                             |
| R         | 旋转                             |
| x         | 删除                             |
| shift + A | 添加模型                         |
| Tab       | 编辑 / 物体                      |
| alt + z   | 透视显示                         |
| ctrl + I  | 反选                             |
| /         | 隔离模式 -- 单独显示当前选择物体 |
| ~         | 视图菜单                         |
| shift + z | 线框模式                         |
| >         | 轴心点菜单                       |
